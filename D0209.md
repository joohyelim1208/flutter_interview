A. 직렬화 역직렬화는 객체의 데이터를 외부와 주고받기 위해 형태를 변환하는 과정입니다.
직렬화는 객체를 JSON문자열(String) 같이 전송 가능한 형태로 변환하는 과정이고 API서버에 데이터를 보낼 때 사용합니다.
역직렬화는 서버에서 받은 JSON이나 데이터를 다시 다트의 객체로 변환하는 것을 말합니다.
(변환 흐름은 직렬화: Dart객체 -> Map(키와 값) -> JSON 문자열. 역직렬화: JSON 문자열 -> Map -> Dart 객체로 변환하는 과정을 거칩니다.

직렬화는 객체에 toJson()메서드를 구현하고 jsonDecode()함수를 사용합니다.
객체 -> Map: toJson(), Map -> Json문자열: jsonDecode()

역직렬화는 jsonDecode()함수로 문자열을 Map으로 변환하는데 fromJson()생성자를 구현합니다.
JSON문자열 -> Map: jsonDecode(), Map -> 객체: fromJson())

객체를 생성해서 Map으로 변환하고 JSON인코딩을 거쳐서 HTTP통신으로 전송한 뒤 역직렬화 과정을 거쳐서 거꾸로 수행하게 됩니다.

이 과정이 필요한 이유는 컴퓨터 메모리 상의 객체의 복잡한 구조 때문에 네트워크로 보내주려면 공통약속인 JSON과 같은 텍스트로 압축해서 보내고 받을 때는 다시 객체로 풀어주는 과정이 필요합니다.

대규모 프로젝트의 경우에는 freezed같은 라이브러리를 사용하는데 빌드러너가 직렬화 코드를 자동으로 생성해주어서 정확하고 안전하며 불변성까지 보장을 해주어서 안전하게 코드를 구현할 수 있습니다.
그리고 Map형태를 직접 사용하기 보다는 model클래스로 변환해서 사용하는 것이 오타와 같은 실수를 줄이고, 타입과 다른 문자열이 들어왔을 때에도 즉시 에러 체크도 가능하며, 
Map을 사용했다면 프로젝트 전체를 검사해서 문자열을 고쳐야 하지만 모델클래스는 내부의 fromJson 한곳만 수정하는 되기 때문에 유지보수에도 편의성을 제공합니다.
